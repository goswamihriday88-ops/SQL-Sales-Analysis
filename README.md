# SQL-Sales-Analysis
# 📊 Sales & Customer Analysis using SQL

## 🎯 Objective
Analyze customer behavior, sales trends, and product performance using SQL.

---

## 🗂️ Dataset
The project uses 3 tables:
- customers
- orders
- retail_sales

---

## 🛠️ Tools Used
- SQL (MySQL / Snowflake compatible)

---

## 📌 Key Analysis Performed

### 1. Customer Analysis
- Total orders per customer
- Customers with no orders

### 2. Sales Segmentation
- Categorized sales into High, Medium, Low

### 3. Top Performers
- Top customer per city using window functions

### 4. Sales Trends
- Day-wise sales comparison using LAG()

### 5. Running Totals
- Cumulative sales per city

---

## 🔥 Key SQL Concepts Used
- Joins (LEFT JOIN)
- Aggregations (SUM, COUNT)
- Window Functions (ROW_NUMBER, LAG, DENSE_RANK)
- CASE Statements
- Subqueries

---

## 📈 Outcome
- Identified top-performing customers and cities
- Analyzed sales growth trends
- Segmented customers based on performance
